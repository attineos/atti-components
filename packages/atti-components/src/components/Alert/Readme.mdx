name: 'Alert'
title: 'Alert component'
---

Alert with error :

```js
<Alert variance="error">
  <Text>There is an error</Text>
</Alert>
```

Alert with success :

```js
<Alert variance="success">
  <Text>Success</Text>
</Alert>
```

Alert with information :

```js
<Alert variance="info">
  <Text>Information</Text>
</Alert>
```

Alert with warning :

```js
<Alert variance="warning">
  <Text>Warning</Text>
</Alert>
```
