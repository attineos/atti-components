Alert component allows you to display various type of information.

## Examples

An error Alert :

```js
<Alert variance="error">
  <Text>There is an error</Text>
</Alert>
```

A success Alert :

```js
<Alert variance="success">
  <Text>Success</Text>
</Alert>
```

An information Alert :

```js
<Alert variance="info">
  <Text>Information</Text>
</Alert>
```

A warning Alert :

```js
<Alert variance="warning">
  <Text>Warning</Text>
</Alert>
```
