Badges are used to display various small pieces of information

## Example

```js
<>
  <Badge variance="danger">Danger</Badge>
  <Badge variance="success">Success</Badge>
  <Badge variance="warning">Warning</Badge>
  <Badge variance="info">Info</Badge>
</>
```
