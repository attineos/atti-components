Breadcrumbs are usually used to navigate across categories and also to help the user to understand where he is.

## Examples

```js
<Breadcrumb separator="-">
  <Breadcrumb.Element href="/#">Home</Breadcrumb.Element>
  <Breadcrumb.Element href="/components">Components</Breadcrumb.Element>
  <Breadcrumb.Element>Breadcrumb</Breadcrumb.Element>
</Breadcrumb>
```

Breacrumb with larger margin and custom separator :

```js
const otherTheme = {
  components: {
    breadcrumb: {
      breadcrumbSeparator: {
        spaces: {
          marginLeft: '16px',
          marginRight: '16px',
        },
      },
    },
  },
}

render(
  <ThemeProvider otherTheme={otherTheme}>
    <Breadcrumb separator={'>'}>
      <Breadcrumb.Element href="/#">Home</Breadcrumb.Element>
      <Breadcrumb.Element href="/components">Components</Breadcrumb.Element>
      <Breadcrumb.Element>Breadcrumb</Breadcrumb.Element>
    </Breadcrumb>
  </ThemeProvider>,
)
```

Thanks to the compound component pattern it's up to the parent element to customize the content of each Breadcrumb.Element component.
Since Breadcrumb is using compound component pattern, renderElement prop is not needed anymore.
If you want to bring specificity based on props, thanks to use composition in order to generate your custom breadcrumb elements.

```js
const Red = styled(Link)`
  color: red;
`
const Green = styled(Link)`
  color: green;
`
const CustomBreadCrumbElement = ({ children, someProps }) => (
  <Breadcrumb.Element>
    {someProps === 10 ? <Green>{children}</Green> : <Red>{children}</Red>}
  </Breadcrumb.Element>
)

render(
  <Breadcrumb separator={'>'}>
    <Breadcrumb.Element>
      <Red href="#">Custom content</Red>
    </Breadcrumb.Element>
    <CustomBreadCrumbElement someProps={10}>Custom Breadcrumb.Element</CustomBreadCrumbElement>
    <CustomBreadCrumbElement someProps={99}>Custom Breadcrumb.Element</CustomBreadCrumbElement>
    <Breadcrumb.Element href="#">Default content with an url</Breadcrumb.Element>
    <Breadcrumb.Element>Default content without an url</Breadcrumb.Element>
  </Breadcrumb>,
)
```
