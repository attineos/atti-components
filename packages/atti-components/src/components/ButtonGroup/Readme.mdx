ButtonGroup can be used for pagination or page size elements.

## Example

```js
<ButtonGroup>
  <Button>Prev</Button>
  <Button>1</Button>
  <Button>2</Button>
  <Button>3</Button>
  <Button>Next</Button>
</ButtonGroup>
```
