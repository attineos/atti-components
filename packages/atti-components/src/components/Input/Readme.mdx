Let the user enter and edit text. This is the basic element of forms.

## Examples

Normal Input type text :

```js
<Input id="inputTestTxt" name="inputTestTxt" placeholder="Input text" />
```

Normal Input type password:

```js
<Input id="inputTest" name="inputTest" placeholder="Password" type="password" />
```
