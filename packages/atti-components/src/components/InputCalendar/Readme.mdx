InputCalendar allows you to pick up a date.

## Examples

Uncontrolled component:

```js
<InputCalendar onChange={date => console.log(date)} />
```

Controlled component:

```js
const Example = () => {
  const [date, setDate] = useState(new Date())
  return <InputCalendar value={date} onChange={date => setDate(date)} />
}

render(<Example />)
```
