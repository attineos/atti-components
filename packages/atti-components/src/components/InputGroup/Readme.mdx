Button group :

Group buttons, such as in pagination or pageSize elements.

```js
<InputGroup>
  <Button>Prev</Button>
  <Button>1</Button>
  <Button>2</Button>
  <Button>3</Button>
  <Button>Next</Button>
</InputGroup>
```

You can also group vertically.

```js
<InputGroup vertical>
  <Button>Prev</Button>
  <Button>1</Button>
  <Button>2</Button>
  <Button>3</Button>
  <Button>Next</Button>
</InputGroup>
```
