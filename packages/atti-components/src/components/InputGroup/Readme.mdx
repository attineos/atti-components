A bit like the ButtonGroup but InputGroup allows you to group even more components such as Inputs.

## Examples

```js
<InputGroup>
  <Button>+</Button>
  <Input type="text" value="1" />
  <Button>-</Button>
</InputGroup>
```

You can also group vertically.

```js
<InputGroup vertical>
  <Button>+</Button>
  <Input type="text" value="1" />
  <Button>-</Button>
</InputGroup>
```
