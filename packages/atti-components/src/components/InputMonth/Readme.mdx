Input Month:

Uncontrolled component:

```js
<InputMonth onChange={date => console.log(date)} />
```

Controlled component:

```js
const Example = () => {
  const [date, setDate] = useState(new Date())
  return <InputMonth value={date} onChange={date => setDate({ date })} />
}

render(<Example />)
```
