Toggle offers the user the possibility to choose or not item.

## Example

```js
<div
  style={{
    paddingBottom: '30px',
    display: 'flex',
    justifyContent: 'space-between',
    margin: '0 48px 0 -10px',
  }}
>
  <div>
    <Toggle id="toggleTest1" name="toggleTestGroup" value="ImChecked" />
  </div>
  <div>
    <Toggle
      id="toggleTest2"
      name="toggleTestGroup"
      value="ImChecked"
      variant="neutral"
      checked="true"
      sized="little"
    />
  </div>
  <div>
    <Toggle
      id="toggleTest3"
      name="toggleTestGroup"
      value="ImChecked"
      variant="disabled"
      click="false"
      sized="medium"
      checked="true"
    />
  </div>
  <div>
    <Toggle
      id="toggleTest4"
      name="toggleTestGroup"
      value="ImChecked"
      variant="on_off"
      sized="medium"
    />
  </div>
</div>
```
