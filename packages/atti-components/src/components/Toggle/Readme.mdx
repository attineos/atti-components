Toggle offers the user the possibility to choose or not item.

## Example

```js
<div
  style={{
    display: 'flex',
    justifyContent: 'space-between',
  }}
>
  <div>
    <Toggle id="toggleTest1" name="toggleTestGroup" value="ImChecked" />
  </div>
  <div>
    <Toggle
      id="toggleTest2"
      name="toggleTestGroup"
      value="ImChecked"
      color="neutral"
      checked="true"
      variant="little"
    />
  </div>
  <div>
    <Toggle
      id="toggleTest3"
      name="toggleTestGroup"
      value="ImChecked"
      color="disabled"
      click="false"
      variant="medium"
      checked="true"
    />
  </div>
  <div>
    <Toggle
      id="toggleTest4"
      name="toggleTestGroup"
      value="ImChecked"
      color="on_off"
      variant="medium"
    />
  </div>
</div>
```
